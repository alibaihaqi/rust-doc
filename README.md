# Rust Documentation

A comprehensive Rust learning documentation site built with VitePress.

## Structure

- `introduction/` - Getting started with Rust
- `beginner/` - Beginner level tutorials building concrete artifacts
- `intermediate/` - Intermediate level tutorials
- `advanced/` - Advanced level tutorials
- `rust-language/` - Deep-dive into Rust language mechanics

## Development

```bash
pnpm install
pnpm docs:dev
```

## Build

```bash
pnpm docs:build
```

## Deploy

Automatically deployed to GitHub Pages on push to main.
