# Beginner Rust Tutorials

Build concrete Rust programs step by step. Each tutorial adds a new concept while producing a working application.

## Learning Path

| # | Tutorial | Concepts |
|---|----------|----------|
| 01 | [Install Rust](./01-install-rust) | Toolchain setup, Cargo basics |
| 02 | [Hello World](./02-hello-world) | Basic syntax, `fn main`, `println!` |
| 03 | [Variables and Mutability](./03-variables-and-mutability) | `let`, `mut`, shadowing, constants |
| 04 | [Data Types](./04-data-types) | Scalars, compounds, type inference |
| 05 | [Functions](./05-functions) | Parameters, returns, statements vs expressions |
| 06 | [Control Flow](./06-control-flow) | `if`, `else`, `loop`, `while`, `for` |
| 07 | [Ownership](./07-ownership) | Move semantics, scope, drop |
| 08 | [References and Borrowing](./08-references-and-borrowing) | `&`, `&mut`, borrow rules |
| 09 | [Structs](./09-structs) | Defining, instantiating, methods |
| 10 | [Enums and Pattern Matching](./10-enums-and-pattern-matching) | `enum`, `match`, `Option`, `Result` |

## Prerequisites

- [Rust installed](../introduction/getting-started)
- Basic programming knowledge (variables, loops, functions)

Start with [01 Install Rust](./01-install-rust).
