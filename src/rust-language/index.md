# Rust Language Deep Dive

Comprehensive tour of Rust's language features from fundamentals to advanced topics.

## Learning Path

| # | Topic | Focus |
|---|-------|-------|
| 01 | [Why Rust](./01-why-rust) | Philosophy, guarantees, use cases |
| 02 | [Variables and Types](./02-variables-and-types) | Type system, inference, mutability |
| 03 | [Ownership and Borrowing](./03-ownership-and-borrowing) | Core memory model |
| 04 | [Structs and Enums](./04-structs-and-enums) | Data modeling, pattern matching |
| 05 | [Pattern Matching](./05-pattern-matching) | `match`, `if let`, guards, `@` bindings |
| 06 | [Traits and Generics](./06-traits-and-generics) | Abstraction, bounds, associated types |
| 07 | [Error Handling](./07-error-handling) | `Result`, `Option`, custom errors |
| 08 | [Testing](./08-testing) | Unit, integration, property-based, benchmarks |
| 09 | [Concurrency](./09-concurrency) | Threads, channels, `Arc`, `Mutex`, `Send`/`Sync` |
| 10 | [Async Programming](./10-async-programming) | Futures, executors, `tokio`, `async-std` |
| 11 | [Standard Library Tour](./11-standard-library) | Collections, I/O, strings, `std` vs `core` |
| 12 | [Macros](./12-macros) | `macro_rules!`, procedural, derive |
| 13 | [Unsafe Rust](./13-unsafe-rust) | When, why, how, soundness |
